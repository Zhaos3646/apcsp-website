Linux Reference Page
Command Description Example
echo Displays a line of text echo Hello World
clear Clears terminal screen clear
yes Repeats a string until killed yes Hello
sleep Delay for a set amount of seconds sleep 5
uptime Display the amount of time the system was running uptime
who Display who is logged in who
last Shows the last logged in user last
date Displays the date date
id Displays the information about a user or group id
history Displays the previously used commands history
